### Common Utilities

Various utility functions and classes for everyday tasks.

#### ResultOf

A powerful alternative to `Result` for handling multiple outcome states (Success, Failure, Warning, Error).

- `ResultOf<T>`
- `ResultOf2<T, V>`
- `ResultOfTripple<T, V, M>`

#### Permutations

Generate all possible combinations of a character set.

- `createPermutations()`: Standard generation.
- `createPermutationsMulti()`: Multi-threaded generation for large sets or ranges.

#### Time Measurement

- `measureTime { ... }`: Measures execution time and returns both the result and the duration.

#### System Properties

- Helpers to access and parse system properties easily.
