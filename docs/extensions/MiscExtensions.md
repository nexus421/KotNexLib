### Miscellaneous Extensions

Small but helpful extensions for basic types.

#### Boolean

- `ifTrue { ... }`: Executes a block if the boolean is true.
- `ifFalse { ... }`: Executes a block if the boolean is false.

#### Number (Int, Double, Float, Long)

- `powOfTwo()`: Returns the square of the number.
- Formatting helpers for better readability.

#### Generic & Other

- `ifNull { ... }`: Provides a fallback or executes code if an object is null.
- `tryOrNull { ... }`: Executes a block and returns null if an exception occurs.
